### Smsbaher-это инструмент для SMS-спама.  Работает на Termux и на Linux.

## Установка
* apt update && apt upgrade;
* pkg install python;
* apt install git;
* git clone https://github.com/oukibattlegod/anton-mayerov.git;
* cd anton-mayerov;
* pip3 install -r requirements.txt;
## Запуск
* python3 smsGOD.py.
